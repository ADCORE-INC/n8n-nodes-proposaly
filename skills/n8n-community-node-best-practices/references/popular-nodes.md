# Popular Community Node Repos (Patterns to Copy)

These are examples of community nodes that demonstrate common patterns (structure, docs, operations):

## firecrawl/n8n-nodes-firecrawl

- Generated from n8n-nodes-starter
- README includes: Installation, Operations, Credentials, Compatibility, Resources, Version history
- Also documents AI Agent tool usage patterns and `$fromAI()` examples

Repo: https://github.com/firecrawl/n8n-nodes-firecrawl

## brave/n8n-nodes-brave-search

- Generated from n8n-nodes-starter
- Clean repo structure: `.github/`, `credentials/`, `nodes/<NodeName>/`
- README has strong “Development” section and local dev workflow

Repo: https://github.com/brave/n8n-nodes-brave-search

## umanamente/n8n-nodes-globals

- Community utility node with clear usage docs and dedicated docs/assets folders
- Shows how non-API nodes can still follow the same packaging and UX conventions

Repo: https://github.com/umanamente/n8n-nodes-globals
